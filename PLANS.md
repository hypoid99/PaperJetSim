# Roadmap — PaperJetSim

## Phase 1 — MVP (목표: 2주)

### Week 1: 물리 엔진 + 렌더링 기반

- [ ] 프로젝트 환경 세팅 (Python venv, requirements.txt)
- [ ] Pyglet 윈도우 + OpenGL 기본 씬 구성
- [ ] 물리 엔진 구현 (`physics/`)
  - [ ] 공기역학 모델 (양력/항력 함수)
  - [ ] 비행기 상태 업데이트 루프
  - [ ] 환경 변수 컨테이너
- [ ] 3D 비행기 모델 렌더링 (단순 종이 형태)
- [ ] 궤적 트레일 렌더링

### Week 2: UI + 게임 루프

- [ ] 발사 컨트롤 UI (각도/힘 슬라이더)
- [ ] 환경 설정 패널
- [ ] 게임 상태 머신 (MENU → SETUP → FLYING → RESULT)
- [ ] HUD 구현
- [ ] SQLite 기록 시스템
- [ ] 기록판 UI
- [ ] 결과 화면 + 신기록 연출

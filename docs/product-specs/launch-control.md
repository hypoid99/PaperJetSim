# 발사 컨트롤 스펙 — PaperJetSim

## 개요

사용자가 종이비행기를 던지기 전 각도와 힘을 조절하는 인터페이스.
직관적이고 즉각적인 피드백이 핵심.

---

## 조절 파라미터

| 파라미터 | 범위 | 기본값 | 단위 |
|---------|------|--------|------|
| 발사 각도 | -10° ~ 45° | 15° | deg |
| 발사 힘 | 0 ~ 100 | 50 | % |

### 발사 속도 변환

```
초기 속도(m/s) = 3.0 + (force / 100) × 12.0
→ 최소: 3 m/s, 최대: 15 m/s
```

## UI 설계

### 화면 레이아웃 (발사 준비 화면)

```
┌──────────────────────────────────────────────┐
│                 3D 뷰포트                      │
│                                              │
│        [종이비행기 3D 모델 미리보기]            │
│              (각도 실시간 반영)                │
│                                              │
├──────────────────────────────────────────────┤
│  각도: [━━━━━●──────] 15°   힘: [━━━━━━●───] 50% │
│                                              │
│  바람: ↗ 2.5 m/s   중력: 9.81 m/s²           │
│                                              │
│         [  발사!  ]    [ 환경 설정 ]           │
└──────────────────────────────────────────────┘
```

### 조작 방법

| 조작 | 동작 |
|------|------|
| 마우스 드래그 (각도 슬라이더) | 발사 각도 조절 |
| 마우스 드래그 (힘 슬라이더) | 발사 힘 조절 |
| 마우스 휠 | 각도 미세 조절 (±1°) |
| `Space` | 발사 |
| `R` | 초기화 (기본값으로) |
| `E` | 환경 설정 패널 토글 |

---

## 시각적 피드백

- **각도 가이드라인**: 3D 뷰에서 발사 예상 초기 궤적을 반투명 선으로 표시
- **파워 표시**: 슬라이더 색상이 힘에 따라 파랑 → 빨강으로 변화
- **각도 숫자**: 슬라이더 위에 실시간 각도값 표시

---

## 구현 파일

- `ui/launch_panel.py` — 슬라이더 위젯, 발사 버튼
- `game/state.py` — SETUP 상태에서 발사 이벤트 처리

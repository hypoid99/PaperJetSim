# 환경 변수 스펙 — PaperJetSim

## 개요

비행에 영향을 주는 외부 환경 조건을 사용자가 직접 조절할 수 있다.
사전 설정된 "날씨 프리셋"으로 빠르게 선택하거나 직접 커스텀할 수 있다.

---

## 환경 변수 목록

| 변수 | 범위 | 기본값 | 게임 영향 |
|------|------|--------|-----------|
| 바람 세기 | 0 ~ 10 m/s | 0 | 강할수록 비행기가 밀려남 |
| 바람 방향 | 0 ~ 360° | 0 (정면) | 측풍은 Z축 편향 발생 |
| 중력 | 1.0 ~ 20.0 m/s² | 9.81 | 낮을수록 더 멀리 날아감 |
| 공기밀도 | 0.5 ~ 1.5 kg/m³ | 1.225 | 높을수록 양력·항력 모두 증가 |
| 돌풍 확률 | 0 ~ 50% | 0 | 비행 중 랜덤 바람 변동 |

---

## 날씨 프리셋

| 프리셋 이름 | 바람 | 중력 | 공기밀도 | 특징 |
|------------|------|------|---------|------|
| 🌤 평온한 날 | 0 m/s | 9.81 | 1.225 | 기본 조건 |
| 💨 순풍 | 3 m/s (정면) | 9.81 | 1.225 | 비행기 뒤에서 밀어줌 |
| 🌬 강풍 | 7 m/s (측풍) | 9.81 | 1.0 | 비행 경로 크게 왜곡 |
| 🏔 고산지대 | 0 m/s | 9.0 | 0.7 | 공기 희박, 양력 감소 |
| 🌙 달 표면 | 0 m/s | 1.62 | 0.0 | 공기 없음, 포물선 낙하 |
| ⚡ 챌린지 | 5 m/s (역풍) | 12.0 | 1.5 | 극한 조건 |

---

## 돌풍 시스템

비행 중 랜덤 돌풍이 발생해 추가 재미 요소를 제공한다.

```
돌풍 이벤트:
- 지속시간: 0.3 ~ 1.0초
- 강도: 설정 바람의 50% ~ 200%
- 방향: 설정 방향 ± 45°
- 시각 효과: 화면 가장자리 깜박임, 바람 파티클
```

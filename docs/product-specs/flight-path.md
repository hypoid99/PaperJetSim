# 비행 경로 시각화 스펙 — PaperJetSim

## 개요

비행 중 실시간으로 3D 공간에 궤적을 그리고,
착지 후에도 전체 경로를 확인할 수 있도록 한다.

---

## 3D 씬 구성

### 카메라

| 모드 | 설명 | 전환 키 |
|------|------|---------|
| 추적 카메라 (기본) | 비행기를 뒤에서 따라가며 촬영 | `C` |
| 사이드뷰 | 옆에서 보는 2.5D 시점 | `C` 반복 |
| 탑뷰 | 위에서 내려다보는 시점 | `C` 반복 |
| 자유 카메라 | 마우스로 자유롭게 회전/이동 | `F` |

### 지형 및 배경

- **바닥**: 끝없이 이어지는 잔디밭 텍스처 평면 (Z-fighting 방지용 약간 두께)
- **하늘**: 단색 그라데이션 또는 스카이박스
- **거리 마커**: 10m 간격 흰 점선, 100m마다 숫자 표시
- **그림자**: 비행기 아래 간단한 blob shadow

---

## 궤적 렌더링

### 실시간 트레일

```
- 매 프레임 비행기 위치를 버퍼에 추가
- 최근 N개 점을 GL_LINE_STRIP으로 렌더링
- 색상: 속도에 따라 파랑(느림) → 노랑 → 빨강(빠름)
- 궤적 두께: 2px
- 최대 포인트: 3600개 (60fps × 60초)
```

### 착지 후 전체 경로 표시

- 비행 종료 후 전체 궤적을 반투명하게 유지
- 최고 높이 지점에 ★ 마커 표시
- 착지 지점에 핀 + 거리 레이블 표시

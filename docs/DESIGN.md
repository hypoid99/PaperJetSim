# Design — PaperJetSim

UI/UX 가이드라인과 시각 디자인 원칙을 정의한다.
Pyglet 기반 데스크톱 게임에 맞춰 작성되었다.

---

## 1. 디자인 원칙

**"던지기 전보다 던진 후가 더 아름다워야 한다."**
발사 전 설정 화면은 단순하게, 비행 중 시각 연출은 풍부하게.

- **즉시성**: 슬라이더를 움직이면 3D 뷰가 즉시 반응한다. 확인 버튼 없이.
- **집중**: 비행 중에는 HUD 외에 UI 요소가 화면에 없다. 비행기만 본다.
- **보상**: 신기록 달성 시 연출이 평범한 착지와 확연히 다르다.

## 4. 화면 레이아웃

### 기본 해상도: 1280 × 720

```
┌────────────────────────────────────────────────┐
│  [HUD 상단 좌측]           [HUD 상단 우측]      │  ← HUD 영역 (80px)
│  거리: 23.5m               기록: 41.3m          │
├────────────────────────────────────────────────┤
│                                                │
│                                                │
│              3D 뷰포트                          │  ← 메인 영역
│                                                │
│                                                │
├────────────────────────────────────────────────┤
│  [발사 컨트롤 패널]                             │  ← 컨트롤 영역 (120px)
│  각도: ━━━━●──  힘: ━━━━━━●  [발사] [환경설정] │
└────────────────────────────────────────────────┘
```

### 반응형 처리

창 크기를 변경해도 비율을 유지한다. 최소 해상도: 800 × 600.

```python
def on_resize(self, width, height):
    self.viewport = (0, 0, width, height)
    self.hud.reflow(width, height)
    self.control_panel.reflow(width, height)
```

## 5. 애니메이션 & 연출

| 상황 | 연출 | 지속시간 |
|------|------|---------|
| 발사 | 카메라 살짝 흔들림 (shake) | 0.2초 |
| 비행 중 돌풍 | 화면 가장자리 파랑 빛 | 0.5초 |
| 착지 (일반) | 먼지 파티클 | 0.8초 |
| 착지 (신기록) | 금색 파티클 + 화면 플래시 | 1.5초 |
| 신기록 텍스트 | 위에서 떨어지는 애니메이션 | 0.6초 |
